# Gleam-ACM-Chapter

set it up

```
  mkdir presentation
  cd ./presentation
  git clone https://github.com/alexandermeade/Gleam-ACM-Chapter.git
```

to run one of the slides from the terminal<br/>
```
  cd ./name_of_the_slide
  gleam run
```
